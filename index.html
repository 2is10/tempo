Simplate

Simplate is a tiny JSON rendering engine that enables you to craft data templates in pure HTML. Never document.write() again.

  • Download
  • Demo
  • GitHub
  • Google Code
  • Works in Safari, Chrome, FireFox, Opera, and Internet Explorer 6+
  • Itty-bitty footprint (2k), lightning-fast rendering (30k of JSON in 5 milliseconds)
  • Works with or without jQuery
  • Degrades gracefully if JavaScript is not enabled
  • Write HTML in HTML documents, not in JavaScript

1. Include the Simplate script

<script src="../src/simplate.js" type="text/javascript"></script>

2. Compose the data template in HTML

<ol id="tweets">
	<li data-template>
		<img src="{{profile_image_url}}" />
		<h3>{{from_user}}</h3>
		<p>{{text}}</p>
	</li>
</ol>

3. Pass in the template ID and the JSON data source

SIMPLATE.prepare("tweets").render( data );

4. Booyah!

  1. {{from_user}}

    {{text}}

Usage

JSON

Simplate takes information encoded as JSON and renders it according to an HTML template. Below is a sample array of JSON data.

var data = [
	{'name':{'first':'Leonard', 'last':'Marx'}, 'nickname':'Chico', 'born': 'March 21, 1887', 'actor': true, 'sole_endevors' : [{'title':'Papa Romani'}]},
	{'name':{'first':'Adolph', 'last':'Marx'}, 'nickname':'Harpo', 'born': 'November 23, 1888', 'actor': true, 'sole_endevors' : [{'title':'Too Many Kisses', 'rating':'favorite'}, {'title':'Stage Door Canteen'}]},
	{'name':{'first':'Julius Henry', 'last':'Marx'}, 'nickname':'Groucho', 'born': 'October 2, 1890', 'actor': true, 'sole_endevors' : [{'title':'Copacabana'}, {'title':'Mr. Music', 'rating':'favorite'}, {'title':'Double Dynamite'}]},
	{'name':{'first':'Milton', 'last':'Marx'}, 'nickname':'Gummo', 'born': 'October 23, 1892'},
	{'name':{'first':'Herbert', 'last':'Marx'}, 'nickname':'Zeppo', 'born': 'February 25, 1901', 'actor': true, 'sole_endevors' : [{'title':'A Kiss in the Dark'}]}
];

JavaScript

Include script

You only need to include a single, 2kb script. You can either download and host the script yourself, or simply reference the minified script on Google Code.

<script src="../src/simplate.js" type="text/javascript"></script>

SIMPLATE.prepare()

To initialize Simplate, run the prepare() function as soon as possible (such as on document.ready). The prepare function will scan the page for data templates, cache them in memory, and remove the data template HTML elements from the page.

element

Tell Simplate the ID of the HTML element containing your data template. If you're using jQuery, you may pass in a jQuery object instead.

SIMPLATE.render()

Once the JSON data is available, run the render() function to add the data to the page.

data

You'll also need to tell Simplate what JSON data you would like to be rendered with the data template. You'll first need to perform an AJAX call to the JSON data source (see below).

SIMPLATE.prepare( element ).render( data );

SIMPLATE.prepare('marx-brothers').render(data);

HTML

data-template

Any tag with a data-template attribute will be considered a data template.

{{fields}}

Any field represented in the JSON data may be retrieved by referencing the field name inside double brackets. 

<ol id="marx-brothers">
	<li data-template>{{nickname}} {{name.last}}</li>
</ol>

The above example would be rendered as:

  1. Chico Marx
  2. Harpo Marx
  3. Groucho Marx
  4. Gummo Marx
  5. Zeppo Marx

Nested data-templates

Simplate even allows you to nest data templates within other data templates. 

<li data-template>
    {{nickname}} {{name.last}}
    <ul>
        <li data-template="sole_endevors">{{title}}</li>
    </ul>
</li>
  1. Chico Marx
    • Papa Romani
  2. Harpo Marx
    • Too Many Kisses
    • Stage Door Canteen
  3. Groucho Marx
    • Copacabana
    • Mr. Music
    • Double Dynamite
  4. Gummo Marx
  5. Zeppo Marx
    • A Kiss in the Dark

Conditionals

Simplate provides boolean and value-based conditionals, as well as the ability to define multiple data templates per container (the first matching template wins).

<ul id="marx-brothers">
	<li data-template data-if-nickname="Groucho"">{{nickname}} (aka {{name.first}}) was grumpy!</li>
	<li data-template data-if-actor>{{name.first}}, nicknamed '<i>{{nickname}} {{name.last}}</i>' was born on {{born}}</li>
	
	<!-- Default template -->
	<li data-template>{{name.first}} {{name.last}} was not in any movies!</li>
</ul>
  • Leonard, nicknamed 'Chico Marx' was born on March 21, 1887
  • Adolph, nicknamed 'Harpo Marx' was born on November 23, 1888
  • Groucho (aka Julius Henry) was grumpy!
  • Milton Marx was not in any movies!
  • Herbert, nicknamed 'Zeppo Marx' was born on February 25, 1901

Using Simplate with jQuery

The jQuery JavaScript library provides any easy method to retrieve JSON via an AJAX request.

var twitter = SIMPLATE.prepare('jquery-ajax-twitter');
$.getJSON("http://search.twitter.com/search.json?q='marx brothers'&callback=?", function(data) {
	twitter.render(data.results);
});
