Simplate

Simplate is a tiny JSON rendering engine that enables you to craft data templates in pure HTML. Never document.write() again.

  • Download
  • Demo
  • GitHub
  • Google Code
  • Works in Safari, Chrome, FireFox, Opera, and Internet Explorer 6+
  • Itty-bitty footprint (2k), lightning-fast rendering (30k of JSON in 5 milliseconds)
  • Works with or without jQuery
  • Degrades gracefully if JavaScript is not enabled
  • Write HTML in HTML documents, not in JavaScript

1. Include the Simplate script

<script src="../src/simplate.js" type="text/javascript"></script>

2. Compose the data template in HTML

<ol id="tweets">
	<li data-template>
		<img src="{{profile_image_url}}" />
		<h3>{{from_user}}</h3>
		<p>{{text}}</p>
	</li>
</ol>

3. Pass in the template ID and the JSON data source

SIMPLATE.prepare("tweets").render( data );

4. Booyah!

  1. {{from_user}}

    {{text}}

JSON

Simplate takes information encoded as JSON and renders it according to an HTML template. Below is a sample array of JSON data.

var data = [
	{'name':{'first':'Leonard', 'last':'Marx'}, 'nickname':'Chico', 'born': 'March 21, 1887', 'actor': true},
	{'name':{'first':'Adolph', 'last':'Marx'}, 'nickname':'Harpo', 'born': 'November 23, 1888', 'actor': true},
	{'name':{'first':'Julius Henry', 'last':'Marx'}, 'nickname':'Groucho', 'born': 'October 2, 1890', 'actor': true},
	{'name':{'first':'Milton', 'last':'Marx'}, 'nickname':'Gummo', 'born': 'October 23, 1892'},
	{'name':{'first':'Herbert', 'last':'Marx'}, 'nickname':'Zeppo', 'born': 'February 25, 1901', 'actor': true}
];

JavaScript

Include script

You only need to include a single, 2kb script. You can either download and host the script yourself, or simply reference the minified script on Google Code.

<script src="../src/simplate.js" type="text/javascript"></script>

SIMPLATE.prepare()

To initialize Simplate, run the prepare() function as soon as possible (such as on document.ready). The prepare function will scan the page for data templates, cache them in memory, and remove the data template HTML elements from the page.

element

Tell Simplate the ID of the HTML element containing your data template. If you're using jQuery, you may pass in a jQuery object instead.

SIMPLATE.render()

Once the JSON data is available, run the render() function to add the data to the page.

data

You'll also need to tell Simplate what JSON data you would like to be rendered with the data template. You'll first need to perform an AJAX call to the JSON data source (see below).

SIMPLATE.prepare( element ).render( data );

SIMPLATE.prepare('simple-marx-brothers').render(data);

HTML

data-template

Any tag with a data-template attribute will be considered a data template. No value for the attribute is needed if the entire dataset should be considered for this template. However, you can choose to use only a subset of the data by specifying the desired JSON field name in the value of the data-template attribute.

<div id="container">
	<span data-template>{{foo.bar}}</span>
</div>

<ul id="more-marx-brothers">
	<li data-template data-if-nickname="Groucho" class="brother {{nickname}}">{{nickname}} (aka {{name.first}}) was grumpy!</li>
	<li data-template data-if-actor class="brother">{{name.first}}, nicknamed '<i>{{nickname}} {{name.last}}</i>' was born on {{born}}</li>
	<!-- Default template -->
	<li data-template id="{{nickname}}" class="brother">{{name.first}} {{name.last}} was not in any movies!</li>
</ul>

Usage

JavaScript

				/*
				 * element : ID or an instance of an HTMLElement
				 * data    : An array of objects

				SIMPLATE.prepare( element ).render( data );

HTML

				<div id="container">
					<span data-template>{{foo.bar}}</span>
				</div>

JavaScript

				SIMPLATE.prepare('simple-marx-brothers').render(data);

				<ol id="simple-marx-brothers">
					<li data-template>{{nickname}} {{name.last}}</li>
				</ol>

Output

  1. {{nickname}} {{name.last}}

Template selectors

JavaScript

				SIMPLATE.prepare('more-marx-brothers').render(data);

HTML

				<style>
					li.brother.Groucho { color: red; }
					li#Gummo {text-decoration: line-through;}
				</style>

				<ul id="more-marx-brothers">
					<li data-template data-if-nickname="Groucho" class="brother {{nickname}}">{{nickname}} (aka {{name.first}}) was grumpy!</li>
					<li data-template data-if-actor class="brother">{{name.first}}, nicknamed '<i>{{nickname}} {{name.last}}</i>' was born on {{born}}</li>

				</ul>

  • {{nickname}} (aka {{name.first}}) was grumpy!
  • {{name.first}}, nicknamed '{{nickname}} {{name.last}}' was born on {{born}}
  • {{name.first}} {{name.last}} was not in any movies!

More complex templates

JavaScript

				SIMPLATE.prepare('complex-marx-brothers').render(data);

HTML

				<table id="complex-marx-brothers" border="0" cellspacing="2" cellpadding="2">
					<tbody>
						<tr>
							<th>Last Name</th>
							<th>First Name</th>
							<th>DOB</th>
							<th>Grumpy</th>
						</tr>
						<tr data-template>
							<td>{{name.last}}</td>
							<td>{{name.first}}</td>
							<td>{{born}}</td>
							<td>{{if nickname == 'Groucho'}}<span>X</span>{{endif}}</td>
						</tr>
					</tbody>
				</table>

Output

Last Name First Name DOB Grumpy
{{name.last}} {{name.first}} {{born}} {{if nickname == 'Groucho'}}X{{endif}}

Using with jQuery for AJAX

JavaScript

				var twitter = SIMPLATE.prepare('jquery-ajax-twitter');
				$.getJSON("http://search.twitter.com/search.json?q='marx brothers'&callback=?", function(data) {
						twitter.render(data.results);
					});

HTML

				<style>
					.tweet img { float: left; height: 48px; width: 48px;}
					.tweet div { margin-left: 58px;}
				</style>

				<div id="jquery-ajax-twitter">
					<div data-template class="tweet">
						<img src="{{profile_image_url}}" />
						<div>
							<a href="http://twitter.com/{{from_user}}">{{from_user}}</a> ({{created_at}})
							<p>{{text}}</p>
						</div>
					</div>
				</div>

Output

{{from_user}} ({{created_at}})
