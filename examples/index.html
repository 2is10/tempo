Simplate

Usage

JavaScript

	/*
	 * element : ID or an instance of an HTMLElement
	 * data    : An array of objects
	 */
	SIMPLATE.prepare( element ).render( data );

HTML

Any tag with data-template attribute will be considered a template

	<div id="container">
		<span data-template>{{foo.bar}}</span>
	</div>

Simple Example (static array of items)

Code

	var template = SIMPLATE.prepare('people');

	template.render([
			{'owner':{'first':'Stefan', 'last':'Olafsson'}, 'nationality': 'Icelandic', 'tired': true},
			{'owner':{'first':'Tyler', 'last':'Tate'}, 'nationality': 'American'},
			{'owner':{'first':'Bjarki', 'last':'Holm'}, 'nationality': 'Icelandic', 'type':'Academic'},
		]);

	<ul id="people">
		<li style="list-style: none; font-weight: bold; border-bottom: 1px solid #000;">List of people (stays here)</li>
		<li data-template data-where-tired class="person">Tired guy that {{owner.first}}</li>
		<li data-template data-where-type="Academic" class="person">Greetings Dr. {{owner.last}} ({{nationality}})</li>
		<li data-template class="person">Hi {{owner.first}} {{owner.last}}</li>
	</ul>

Output

  • List of people (stays here)
  • Tired guy that {{owner.first}}
  • Greetings Dr. {{owner.last}} ({{nationality}})
  • Hi {{owner.first}} {{owner.last}}
