Simplate > The Simple JavaScript Template Engine

Usage

JavaScript

	/*
	 * element : ID or an instance of an HTMLElement
	 * data    : An array of objects
	 */
	SIMPLATE.prepare( element ).render( data );

HTML

Any tag with data-template attribute will be considered a template

	<div id="container">
		<span data-template>{{foo.bar}}</span>
	</div>

Examples

The following examples will use this sample data

Simple template

JavaScript

	SIMPLATE.prepare('simple-marx-brothers').render(data);

HTML

	<ol id="simple-marx-brothers">
		<li data-template>{{nickname}} {{name.last}}</li>
	</ol>

Output

  1. {{nickname}} {{name.last}}

Nested templates

JavaScript

	SIMPLATE.prepare('nested-marx-brothers').render(data);

HTML

Output

  1. {{nickname}} {{name.last}}
    • {{title}}

Template selectors

JavaScript

	SIMPLATE.prepare('more-marx-brothers').render(data);

HTML

	<style>
		li.brother.Groucho { color: red; }
		li#Gummo {text-decoration: line-through;}
	</style>

	<ul id="more-marx-brothers">
		<li data-template data-if-nickname="Groucho" class="brother {{nickname}}">{{nickname}} (aka {{name.first}}) was grumpy!</li>
		<li data-template data-if-actor class="brother">{{name.first}}, nicknamed '<i>{{nickname}} {{name.last}}</i>' was born on {{born}}</li>
		<!-- Default template -->
		<li data-template id="{{nickname}}" class="brother">{{name.first}} {{name.last}} was not in any movies!</li>
	</ul>

Output

  • {{nickname}} (aka {{name.first}}) was grumpy!
  • {{name.first}}, nicknamed '{{nickname}} {{name.last}}' was born on {{born}}
  • {{name.first}} {{name.last}} was not in any movies!

More complex templates

JavaScript

	SIMPLATE.prepare('complex-marx-brothers').render(data);

HTML

	<table id="complex-marx-brothers" border="0" cellspacing="2" cellpadding="2">
		<tbody>
			<tr>
				<th>Last Name</th>
				<th>First Name</th>
				<th>DOB</th>
				<th>Grumpy</th>
				<th>Solo stuff</th>
			</tr>
			<tr data-template>
				<td>{{name.last}}</td>
				<td>{{name.first}}</td>
				<td>{{born}}</td>
				<td>{{if nickname == 'Groucho' && name.last == 'Marx'}}<span>X</span>{{endif}}</td>
				<td>{{sole_endevors}}</td>
			</tr>
		</tbody>
	</table>

Output

Last Name First Name DOB Grumpy
{{name.last}} {{name.first}} {{born}} {{if nickname == 'Groucho' && name.last == 'Marx'}}X{{endif}}

Using with jQuery for AJAX

JavaScript

	var twitter = SIMPLATE.prepare('jquery-ajax-twitter');
	$.getJSON("http://search.twitter.com/search.json?q='marx brothers'&callback=?", function(data) {
			twitter.render(data.results);
		});

HTML

	<style>
		.tweet img { float: left; height: 48px; width: 48px;}
		.tweet div { margin-left: 58px;}
	</style>

	<div id="jquery-ajax-twitter">
		<div data-template class="tweet">
			<img src="{{profile_image_url}}" />
			<div>
				<a href="http://twitter.com/{{from_user}}">{{from_user}}</a> ({{created_at}})
				<p>{{text}}</p>
			</div>
		</div>
	</div>

Output

{{from_user}} ({{created_at}})

{{text}}
